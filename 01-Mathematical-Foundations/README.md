# 01 - Mathematical Foundations 数学基础

## 📋 模块概述

数学是AI的基石。这个模块涵盖线性代数、概率论、优化理论等核心数学知识。

## 📚 学习内容

### 线性代数（Linear Algebra）
- 向量与矩阵运算
- 矩阵分解（SVD, EVD）
- 线性变换
- 特征值与特征向量

**应用场景**：
- 词嵌入（Word2Vec, GloVe）
- 降维（PCA）
- 推荐系统

### 概率论（Probability Theory）
- 概率基础
- 条件概率与贝叶斯定理
- 常见分布（高斯分布、伯努利分布）
- 期望与方差

**应用场景**：
- 统计机器翻译
- 生成模型
- 不确定性建模

### 优化理论（Optimization）
- 梯度下降及其变体
- 凸优化
- 拉格朗日乘数法
- KKT条件

**应用场景**：
- 神经网络训练
- 参数调优
- 损失函数最小化
