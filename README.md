## 📖 关于本仓库

这是我的AI学习记录仓库，记录了从数学基础到深度学习应用的完整学习路径。作为一个跨界学习者（影视后期 × AI技术），我希望将艺术创作与技术能力结合，探索AI在创意产业中的应用。

## 🎯 学习目标

- **短期目标（6个月）**：建立AI应用判断力，完成课程学分
- **长期目标**：成为能够独立开发AI工具的技术艺术家
- **不是目标**：成为技术架构大师（够用就好）
