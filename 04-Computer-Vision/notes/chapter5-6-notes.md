## 第 5 章 图像复原与重建

### 5.1 图像退化/复原模型

**核心思想：建立"图像是怎么变差的"模型，然后反向修复。**

```plaintext
退化模型：
原图 f(x,y) → 退化函数 H → 噪声 η → 观测图像 g(x,y)

复原模型：
观测图像 g(x,y) → 复原滤波器 → 近似原图 f̂(x,y)
```

**公式：**\
$g(x,y) = H[f(x,y)] + \eta(x,y)$

其中：
- $f(x, y)$ = 原始图像
- $H$ = 退化函数（模糊、运动等）
- $\eta$ = 噪声
- $g(x, y)$ = 观测到的图像

---

### 5.2 噪声模型

**核心问题：图像中的"噪点"是从哪来的？**

#### 常见噪声类型

| 噪声类型 | 特点 | 来源 |
| --- | --- | --- |
| 高斯噪声 | 服从正态分布 | 电子传感器 |
| 椒盐噪声 | 随机黑白点 | 传感器故障、传输错误 |
| 周期噪声 | 重复条纹 | 电气干扰 |
| 均匀噪声 | 均匀分布 | 量化 |

#### 噪声可视化

```plaintext
高斯噪声：       椒盐噪声：       周期噪声：
████████████    ██░░████████    █░█░██░█░█
████████████    ░░██████████    █░█░██░█░█
（有随机噪点）    ↑随机黑白点      ↑周期性条纹
```

---

### 5.3 空间滤波（去噪）

**适用于：噪声单独存在，没有模糊。**

#### （1）均值滤波器

| 滤波器 | 核 | 特点 |
| --- | --- | --- |
| 算术均值 | 全部 1/mn | 简单，但会使边缘模糊 |
| 几何均值 | 乘积开方 | 保留更多细节 |
| 谐波均值 | 倒数平均 | 对盐噪声有效 |
| 逆谐波均值 | 可选 | 对椒盐都有效 |

**算术均值滤波器：**\
$\hat{f}(x,y) = \frac{1}{mn} \sum_{(s,t) \in S_{xy}} g(s,t)$

#### （2）中值滤波器

**对椒盐噪声最有效！**

```plaintext
原图：           中值滤波后：
████████████    ████████████
██■□□□□■██  →  ████████████
████████████    ████████████
↑椒盐噪声        ↑被消除
```

**原理：取邻域像素的中值，噪点被"中和"**

#### （3）自适应滤波器

根据局部特征自动调整滤波强度，效果更好但计算复杂。

---

### 5.4 频率域滤波（去周期噪声）

**适用于：周期性噪声（如条纹）**

#### 陷波滤波器

**核心思想：在频率域"挖掉"特定的噪声频率**

```plaintext
陷波滤波器工作：
频谱图：         陷波后：
●●●●●●●        ●       ●
●●●●●●●●      ●       ●
●●■●●●●●  →  ●   ✕   ●  ← 挖掉噪声点
●●●●●●●●      ●       ●
●●●●●●●        ●       ●
```

### 5.5 线性位置不变退化

**假设：退化过程是线性的，且位置不变。**

**公式：**\
$g(x,y) = h(x,y) * f(x,y) + \eta(x,y)$

其中 $h(x, y)$ 是点扩散函数（PSF）。

---

### 5.6 估计退化函数

**问题：我们不知道退化函数 H，怎么修复？**

#### 三种估计方法

| 方法 | 说明 |
| --- | --- |
| 图像观察估计 | 从退化图像本身推测 |
| 试验估计 | 用已知输入获得输出，推测 H |
| 数学建模 | 用物理模型（如运动模糊公式） |

---

### 5.7 逆滤波

**最直接的复原方法：**

$\hat{F}(u,v) = \frac{G(u,v)}{H(u,v)}$

**问题：** 当 $H(u, v)$ 很小时，噪声会被放大！

---

### 5.8 维纳滤波（最重要！）

**核心思想：在去模糊和抑制噪声之间找平衡。**

$\hat{F}(u,v) = \left[ \frac{1}{H(u,v)} \cdot \frac{|H(u,v)|^2}{|H(u,v)|^2 + \frac{S_\eta(u,v)}{S_f(u,v)}} \right] G(u,v)$

其中：
- $S_\eta$ = 噪声功率谱
- $S_f$ = 图像功率谱
- $\frac{S_\eta}{S_f}$ = 噪声与信号的功率比

**简化理解：**

$H_{wiener} = \frac{H^*}{|H|^2 + K}$

- $K$ = 噪声与信号功率比
- $K$ 大 → 更注重抑制噪声
- $K$ 小 → 更注重去模糊

---

### 5.9 约束最小二乘滤波

**更稳定的复原方法，对噪声更鲁棒。**

公式较复杂，但核心思想是：
- 引入平滑约束
- 避免噪声放大

---

### 5.10 图像重建（CT 原理）

**核心应用：从投影数据重建图像（CT 扫描的原理）**

#### 核心概念： radon 变换

```plaintext
投影：
从不同角度发射X射线
↖  ↗
  ↘  ↙  物体
    ↓↓↓
   探测器接收

反投影：
把各个角度的投影
"叠加"回去
    ↓↓↓
   重建图像
```

#### 重建方法

| 方法 | 说明 |
| --- | --- |
| 反投影 | 简单但有模糊 |
| 滤波反投影 | 加滤波器后反投影，更清晰 |
| 迭代重建 | 逐次逼近，适合稀疏数据 |

## 第 6 章 彩色图像处理

### 6.1 彩色基础

**为什么需要彩色？**
- 人眼对彩色更敏感
- 彩色信息更丰富
- 符合人类视觉特性

#### 三原色（RGB）

```plaintext
红 + 绿 + 蓝 = 可见光

红   绿   蓝   混合
●    ●    ●
●●   ●●   ●●
+ ●● = ●●  → ●●●●
 ●    ●    ●    ●●
```

#### 互补色

```plaintext
青   = 绿 + 蓝
品红 = 红 + 蓝
黄   = 红 + 绿
```

---

### 6.2 彩色模型

#### （1）RGB 模型

**面向硬件（显示器、相机）**

```plaintext
RGB立方体：
    B(0,0,1)
       │  /│
       │ / │
       │/  │
       /   │
R(1,0,0)┼── G(0,1,0)
       │   │
       │  /
       │ /
       │/
      黑      白

每个点 = (R, G, B)，范围0-255
```

#### （2）CMY/CMYK 模型

**面向印刷（墨水吸收光）**

```plaintext
C = 1 - R  （青   = 白 - 红）
M = 1 - G  （品红 = 白 - 绿）
Y = 1 - B  （黄   = 白 - 蓝）

CMYK：加入黑色K，提高暗部表现
```

#### （3）HSI 模型

**面向人类感知（更直观）**

| 分量 | 含义 | 范围 |
| --- | --- | --- |
| H (Hue) | 色相/颜色 | 0°-360° |
| S (Saturation) | 饱和度/纯度 | 0-100% |
| I (Intensity) | 亮度/明暗 | 0-100% |

```plaintext
HSI模型：
    │
  H │ 从红→黄→绿→青→蓝→品红→红
    │
    ├────────────
  S │ 饱和度：颜色纯度
    │   100% = 纯色
    │   0%   = 灰色
    │
    ├────────────
  I │ 亮度：明暗程度
    │
```

**优势：可以独立处理亮度（I），不影响颜色（H, S）**

---

### 6.3 伪彩色处理

**核心思想：给灰度图像"上色"，便于观察。**

```plaintext
灰度图像：         伪彩色处理：
████████████      ████黄绿红███
████████████  →   ███████████
（看不出差异）     （不同灰度变不同颜色）
```

**应用：**
- 红外图像上色
- 温度图（蓝=冷，红=热）
- 医学图像增强

---

### 6.4 全彩色图像处理

**三种处理方式：**

| 方式 | 说明 |
| --- | --- |
| 分别处理各通道 | RGB 各自处理后合并 |
| 转换到其他空间 | 转 HSI 处理后转回 |
| 向量方法 | 把像素当向量处理 |

---

### 6.5 彩色变换

$g(x,y) = T[f(x,y)]$

| 变换类型 | 说明 |
| --- | --- |
| 色调调节 | 调整整体颜色倾向 |
| 颜色平衡 | 调整红/蓝比例 |
| 饱和度调节 | 颜色的鲜艳程度 |

---

### 6.6 彩色平滑/锐化

#### 彩色平滑

**邻域平均会模糊颜色！**

**解决方法：只在亮度通道平滑**

```plaintext
RGB分别平滑 → 颜色模糊
HSI只在I通道平滑 → 亮度平滑，颜色保持
```

#### 彩色锐化

**在 HSI 空间处理效果更好**

---

### 6.7 彩色边缘检测

**问题：RGB 各通道分别检测会得到不同结果**

**解决：向量梯度方法**

$\Delta \vec{V} = \begin{bmatrix} \frac{\partial R}{\partial x} & \frac{\partial G}{\partial x} & \frac{\partial B}{\partial x} \\ \frac{\partial R}{\partial y} & \frac{\partial G}{\partial y} & \frac{\partial B}{\partial y} \end{bmatrix}^T$

---

### 6.8 彩色分割

#### （1）HSI 空间分割
- H 通道：分割特定颜色
- S 通道：分割饱和度范围
- I 通道：分割亮度范围

#### （2）RGB 空间分割
- 定义"目标颜色"的 RGB 范围
- 判断每个像素是否在范围内

---

### 6.9 彩色图像去噪

| 方法 | 说明 |
| --- | --- |
| 各通道独立去噪 | RGB 分别处理，可能产生色偏 |
| 向量滤波 | 考虑颜色向量整体，效果更好 |
| HSI 空间去噪 | 只在 I 通道去噪，颜色保持 |

---

### 6.10 彩色图像压缩

**压缩原理：利用颜色冗余**
- 人眼对亮度敏感，对颜色不敏感
- 可以减少颜色分辨率（4:2:0 采样）
- JPEG 压缩利用此原理

---

## 公式汇总

| 公式 | 名称 |
| --- | --- |
| $g(x, y) = H[f(x, y)] + \eta$ | 退化模型 |
| $\hat{f}(x, y) = \frac{1}{mn} \sum g(s, t)$ | 算术均值滤波 |
| $\hat{F} = \frac{G}{H}$ | 逆滤波 |
| $\hat{F} = \frac{H^*}{\|H\|^2 + K} G$ | 维纳滤波 |
