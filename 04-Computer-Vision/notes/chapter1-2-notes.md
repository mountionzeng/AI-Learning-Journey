## 第 1 章：引言

### 1.1 什么是数字图像处理？

**核心定义：**
- 图像：二维函数 $f(x, y)$，$x$ 和 $y$ 是空间坐标，$f$ 是强度/灰度
- 数字图像：$x$、$y$、$f$ 都是有限离散量
- 像素（Pixel）:picture element，图像最小单元

```plaintext
图像 = 矩阵
例如 4×4 图像：
[200, 180, 160, 140]
[190, 170, 150, 130]
[180, 160, 140, 120]
[170, 150, 130, 110]
每个数字 = 一个像素值
```

### 图像处理三个层次

## 分层解释

想象你教一个小孩子认识一只猫：

```plaintext
┌─────────────────────────────────────────────────────────┐
│                  高层处理（HIGH-LEVEL）                  │
│                     阶段："理解"                         │
│             输入：特征（这是猫有耳朵、有尾巴...）         │
│              输出：概念/结论（"这是一只猫"）              │
│                                                          │
│              → 就像小孩说："我知道这是猫！"               │
├─────────────────────────────────────────────────────────┤
│                  中层处理（MID-LEVEL）                   │
│                     阶段："看懂"                         │
│                       输入：图像                         │
│          输出：特征/属性（边缘、轮廓、形状、颜色...）     │
│                                                          │
│            → 就像小孩能画出猫的轮廓、说出特点             │
├─────────────────────────────────────────────────────────┤
│                  低层处理（LOW-LEVEL）                   │
│                     阶段："看清"                         │
│                       输入：图像                         │
│                       输出：图像                         │
│                                                          │
│            → 就像小孩先把模糊的照片变清晰                │
└─────────────────────────────────────────────────────────┘
```

---

## 举例说明

**低层处理（输入图像→输出图像）：**
- 照片去噪（手机相机的夜景模式）
- 调亮/调暗
- 锐化（让模糊的细节变清楚）
- 模糊（美颜相机的背景虚化）

```plaintext
原图（模糊） → 锐化处理 → 变清晰
```

**中层处理（输入图像→特征）：**
- 边缘检测（找到物体的边界）
- 分割（把猫从背景中"抠"出来）
- 描述（这只猫是白色的、胖胖的）

```plaintext
整张照片 → 分割 → 猫的区域 + 背景区域
```

**高层处理（输入特征→理解）：**
- 识别（这是猫）
- 判断（这是不是我的猫）
- 行动（如果是陌生人靠近，提醒主人）

```plaintext
猫的特征 → AI识别 → "这是橘猫，体重10斤"
```

## 为什么需要分层？

**分工明确，各司其职：**
1. 低层处理：基础工作——先把图像"整理"干净
2. 中层处理：提取信息——从图像中"抽取"有用特征
3. 高层处理：理解决策——根据特征"做判断"

| 层次 | 输入 | 输出 | 例子 |
|---|---|---|---|
| 低层 | 图像 | 图像 | 去噪、增强 |
| 中层 | 图像 | 属性 | 分割、描述 |
| 高层 | 属性 | 理解 | 识别、推理 |

### 1.2 发展历史

- **1921 年**：Bartlane 电缆传输系统（最早数字图像）
- **1964 年**：Ranger 7 月球图像处理（数字图像处理诞生标志）
- **1970 年代**：CT 发明（革命性医学应用）

### 1.3 应用领域

**电磁波谱成像：**

| 波段 | 应用 |
|---|---|
| γ射线 | 骨扫描、PET、天文 |
| X 射线 | 医学透视、CT、工业检测 |
| 紫外线 | 荧光显微、侦查 |
| 可见光 | 普通摄影、显微镜 |
| 红外线 | 夜视、热成像、遥感 |
| 微波 | 雷达、气象 |
| 无线电 | MRI 核磁共振 |

## 第 2 章：数字图像基础

### 2.1 人眼视觉感知

- 亮度适应：人眼对亮度是**对数感知**
- 同时对比度：同一灰度在不同背景下看起来不同
- 视觉错觉：大脑会"脑补"

### 2.2 光与电磁波谱

**核心公式：**\
$f(x,y) = i(x,y) \cdot r(x,y)$

- $i(x, y)$ = 入射光总量（照明）
- $r(x, y)$ = 反射系数（0-1 之间）
- $f(x, y)$ = 最终亮度

**电磁波谱（从短到长）：**

```plaintext
γ射线 → X射线 → 紫外线 → 可见光 → 红外线 → 微波 → radio波
                              ↑        ↑
                           400nm    700nm
                          （紫）    （红）
```

可见光只是很小的一段！

### 2.3 图像获取

**传感器类型：**
1. 单个传感器：逐点扫描
2. 线传感器：逐行扫描
3. 阵列传感器：CCD/CMOS（一次性拍整个画面）

**获取过程：**

```plaintext
光 → 传感器 → 电信号 → A/D转换 → 数字图像
```

### 2.4 采样与量化（核心！）

**采样 = 空间离散化**

```plaintext
连续：─────────────────
采样：  ●   ●   ●   ●   ●   ●
```

**量化 = 幅度离散化**

```plaintext
连续灰度： 0.0 ──────────── 255.0
量化后：    0    1    2  ...  255
```

**常见分辨率：**
- 空间：1920×1080（Full HD）、3840×2160（4K）
- 灰度：8 位（256 级）、12 位（4096 级）

### 2.5 像素间关系

**邻域：**
- 4 邻域：上下左右 (x±1, y), (x, y±1)
- 8 邻域：周围 8 个点

**连通性：**
- 4-连通：上下左右
- 8-连通：周围 8 个
- m-连通：混合

**距离度量：**

| 距离 | 公式 | 形状 |
|---|---|---|
| 欧氏 | $\sqrt{\Delta x^2 + \Delta y^2}$ | 圆形 |
| 城市 | $\|\Delta x\| + \|\Delta y\|$ | 菱形 |
| 棋盘 | $\max(\|\Delta x\|, \|\Delta y\|)$ | 正方形 |

### 2.6 数学工具基础

需要的基础：
- 线性代数（矩阵运算）
- 概率统计（均值、方差）
- 微积分（连续→离散）
- 向量（梯度、方向）

## 公式汇总

1. **图像定义**：$f(x, y)$ = 二维函数
2. **图像模型**：$f(x, y) = i(x, y) \cdot r(x, y)$
3. **欧氏距离**：$d_e = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}$
4. **城市距离**：$d_4 = |x_1-x_2| + |y_1-y_2|$
5. **棋盘距离**：$d_8 = \max(|x_1-x_2|, |y_1-y_2|)$
