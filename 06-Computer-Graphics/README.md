# 06 - Computer Graphics 计算机图形学

## 📋 模块概述

计算机图形学是影视后期与游戏开发的核心技术。这个模块涵盖渲染管线、着色器编程等CG技术。

## 📚 学习内容

### 渲染管线（Rendering Pipeline）
- 顶点处理
- 光栅化
- 片段着色
- 混合与输出

### 着色器（Shaders）
- Vertex Shader
- Fragment Shader
- Compute Shader
- GLSL/HLSL编程

### 光照模型
- Phong光照模型
- PBR（基于物理的渲染）
- 全局光照（Global Illumination）

- [ ] PBR材质渲染
- [ ] 实时阴影
- [ ] 后期特效（Bloom, DOF）
