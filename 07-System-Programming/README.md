# 07 - System Programming 系统编程

## 📋 模块概述

系统编程是构建高性能应用的基础。这个模块涵盖操作系统、数据库系统等底层技术。

- 内存管理
- 文件系统
- 进程同步（CS 520内容）

### 数据库系统（Database Systems）
- SQL基础
- 索引与优化
- 事务处理
- NoSQL数据库
